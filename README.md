# Aprendizados Novos e Antigos

Este repositório tem como intuito o aprendizado contínuo. Ele serve como um registro de projetos e aprendizados que coloquei em prática, para que eu possa consultar e utilizar futuramente quando necessário.

## O Que Tenho Aprendido

Ao longo do desenvolvimento dos projetos e scripts aqui presentes, tenho aprofundado meus conhecimentos nas seguintes áreas:

- **Análise e Exploração de Dados:** Manipulação avançada de datasets utilizando a linguagem Python, envolvendo bibliotecas fundamentais como Pandas e NumPy para limpeza, formatação e extração de *insights* robustos.
- **Estatística e Machine Learning:** Criação e interpretação de modelos de regressão linear para fazer previsões com base em dados.
- **Integração de APIs e Automação:** Consumo de APIs diversas (como dados de clima, espaço) e construção de bots robustos (por exemplo, um bot para Discord com capacidades de voz, transcrição de áudio e uso da API do Gemini).
- **Banco de Dados e SQL:** Integração direta de dados relacionais dentro de aplicações Python, bem como estruturação e consultas SQL complexas.
- **Engenharia de Software e UI:** Boas práticas de código, versionamento, documentação técnica, e construção de interfaces interativas.

## Último Projeto: Análise de Avaliações de Filmes

Meu projeto mais recente gira em torno da **Análise de Dados de Avaliações de Filmes** (`movie_rating_analytics`).

O que foi feito no projeto:
- Parse, limpeza e processamento de três bases de dados centrais: registros de histórico de usuários, avaliações (`ratings`) e recomendações de filmes.
- Aplicação de técnicas estatísticas para chegar a respostas assertivas e objetivas em cima de questionamentos de negócio pré-definidos.
- Geração automatizada de *outputs* descritivos em texto (ex: `relatorio_analise_dados.md`), permitindo um processo analítico reprodutível e uma leitura técnica limpa para a validação rápida de hipóteses sobre o comportamento do usuário e relevância do catálogo de filmes.
